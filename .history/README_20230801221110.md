# @zzopen/password-manager
管理网址，账号，密码登信息的本地app

An Electron application with Vue and TypeScript


## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
