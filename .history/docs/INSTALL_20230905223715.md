# zz-cipher
一款本地使用的密码管理器

## 特性
- 架构采用electron+go
- sqlite存储密码等敏感记录
- lowdb存储本地数据
- 数据导入导出，云端备份
- 一键复制，省去记忆和填写密码的麻烦
